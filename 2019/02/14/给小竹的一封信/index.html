给小竹的一封信

发表于 2019-02-14 | 分类于 日常

亲爱的小竹竹：

你好！

一直以来都想给你写封信，但是我却缺乏像你一样的说做就做的洒脱，总是想着计划着给自己一个动笔的契机，或者说是一个合适的时间点，也一直拖到了现在。正好，今年借着这个特殊的节日给你写一封短短的信。

信的开头总是不知道如何谈起，这封信的开头我想讲一下我心中的小竹是什么样的。三伏天的夏日，坐在我的书房，空调的冷气能够隔开一切蒸腾的热流，让我用一种更加自由，慵懒的姿态去阅读窗外的景色。每天总有一缕阳光从正午开始就通过狭窄的凹槽，几次折射，跳跃在我的书桌上。看到那个可爱的小光斑在透明的玻璃之上耀动，总有一种让我说不出的开心。每每看到你，也是如此。

小竹是个很念旧的人，用着几年前的苹果手机，家里的娃娃也从来不更新换代只是简简单单的累积，一直很怀念支教的日子，喜欢听老老的歌曲。小竹竹心中是柔软的树袋熊，而不是你自己觉得自己的一个戴着小拳套的袋鼠。

刚开始的时候总会有很多的起起伏伏需要去磨合一下，小竹显得有一些沮丧，不过可爱的小宝宝总是能和我一起调整过来，我们一起做了好多好多事情，一起去吃师大夜市，一起去音乐节，一起在11点的哈工大散散步，一起去看我以前绝对不会看的电影，一起吃热干面，火锅，肯德基，一起去北京和天津旅行，在江边散步，一起开心，一起不开心，一起在下雪的时候啵叽，一起领快递，一起去小旮旯里吃干锅，一起复习CSAPP，一起在机场分开的时候偷偷抹眼泪，不知不觉，我的生活的细枝末节也是小竹的发梢，衣角的毛边，用过的橡皮屑。曾想过以后在老了之前，要去走走这个小小的蔚蓝星球，现在每一张行程表都订好了两张票。我们要在古巴喝啤酒，去挪威的极光下拥抱取暖，去日本北海道吃烤肉，看下雪，去村上的故居跑步，到美国的佛罗里达去喝蓝色玛格丽特和特基拉日出。

在我们的整个恋爱过程中，小竹竹总是很努力的在保持着自己独立而依赖的天性，保持着若有若无的神秘感，可爱又认真的和我计划着好多好多与她小可爱的外表不太相符的宏图远志，尽管自己时不时有些垂头丧气，但是却一直执着的让自己充满热情。小竹很喜欢星空，喜欢把自己和我比作星星。如果你是冥王星的话，我乐意做那个卡戎，挺好的，两个人无人叨扰，不用去管那些什么遥远的地球上的人们在做些什么事情，就连太阳想管住你我都要花些时日，我们只需要在一片虚无中你亲着我，我拉着你，相互凝视，彼此依赖，成为稳定不可分离的双星。

180多天其实根本不算很久，因为我们以后还有很多很多的180天，但是跟小竹腻在一起就感觉两个人竟然有种莫名的契合感，每天清晨的脑回路重新开始运转想到的第一件事就是给小糯米打招呼。日子还要这样跟着沙漏走好久，我们还要一起手牵手走好多个春夏秋冬，不管是在寒冷的哈尔滨或者是以后的天涯海角，只要有小竹的地方，就不会觉得渺远，就会有个锚点。
现在，或者说今天的愿望，就是能在40岁的时候，给你炒一盘苦瓜，两个人就着酒吃，一个人一口，任由清苦的滋味在嘴里扩散开来，接着两双筷子不小心夹起同一片，又同时放开，对视一眼，甜味就能从眼角蔓延到嘴里。苦瓜，又叫半生瓜，年至半生，才能吃出其中的滋味来，期待和你一起吃的那盘半生瓜。

好了，我的小年画，小糯米，小可爱，小乖乖，小宝宝，小仙女，小精灵，小竹竹，小公主
情人节快乐！

大爷留几块银子呗(=^w^=)
